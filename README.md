# wtv
The main objective of the project is the practical application of knowledge gained during laboratory classes in the areas of the Linux system, automation in Bash, text composition in LaTeX, and version control with Git.
Each student is tasked with creating a complete and professional repository on the GitHub platform, which will serve as a portfolio of completed tasks.

The project not only verifies technical skills but also teaches fundamental practices used in the IT industry, such as taking care of change history, project organization, and creating clear documentation.
